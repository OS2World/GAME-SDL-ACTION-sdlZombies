# GAME-SDL-sdlZombies
SdlZombies is a game about tricking the zombies to fall into the holes. Its quite simple at the beginning, but them it gets harder since the zombies start moving faster.

SdlZombies is based on a very old game who run on ZX81 (Sinclair).
